
# Global Cities Dashboard

## Features
- Interactive filters
- KPI cards
- Professional charts
- Responsive layout

## Technologies
- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn

## Run Locally

Install requirements:

```bash
pip install -r requirements.txt
```

Run the dashboard:

```bash
streamlit run app.py
```

## Dataset
Uses the provided dataset file:
`cities500.txt`

Do NOT rename the dataset file.
